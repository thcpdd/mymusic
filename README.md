# 我的音乐

## 介绍
我的音乐Django项目，我的第一个完整的Django项目。

访问地址：<a href="https://mymusic.thcpdd.com">https://mymusic.thcpdd.com</a>

项目完成时间：2023年5月24日

## 项目收获

1. 熟悉了用Django开发一个完整网站的流程。
2. 能独立完成一个完整的网站，虽然大部分是照着已有代码敲的哈哈哈。
